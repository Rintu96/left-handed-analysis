# 🧠 Do Left-Handed People Die Earlier?

## 📌 Overview
This project analyzes whether left-handed individuals have a shorter lifespan using probability and real-world data.

## 🔧 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## 📊 Key Steps
- Data cleaning and preprocessing
- Feature engineering (Birth year, Mean handedness)
- Probability modeling using Bayes' Theorem
- Visualization of age distributions

## 📈 Results
- In 1990 → noticeable lifespan difference
- In 2018 → ~0 difference (-0.03 years)

## 🔥 Conclusion
The apparent lifespan difference is a statistical illusion caused by generational changes in reported handedness.
